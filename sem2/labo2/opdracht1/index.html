Opdracht 1

Confirm:

Wat is de return value van de confirm functie als de gebruiker op een van de buttons klikt?
Als hij op "Ok" klikt returned de functie true ander returned die false.

Prompt:

Wat is de return value van de prompt functie als de gebruiker een tekst intypt en op "ok" drukt?
Dan krijgt de functie de ingevulde tekst terug als return waarde.

Wat is de return value van de prompt functie als de gebruiker op "cancel" klikt
Dan krijgt de functie "null" terug als return waarde.
